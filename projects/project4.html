Developed computer vision and graph-based analysis tools for 3D histology imaging of the vagus nerve, a key target for electrical stimulation therapies. The goal was to automate segmentation, fiber tracking, and error detection, enabling researchers to better understand how stimulation parameters affect nerve function.

Problem

Manual tracing of nerve fibers from histological images is slow and error-prone, often taking hours per patient. Additionally, segmentation algorithms can misclassify nerve regions, requiring labor-intensive corrections. The goal was to build automated, data-driven pipelines to accelerate analysis and improve reproducibility.

Approach

  • Segmentation Analysis: Applied OpenCV‚Äôs connectedComponents algorithm to identify distinct nerve profiles in each image slice and separate the main trunk from branching structures.
  • Graph Construction: Connected fibers between frames using area overlap metrics and converted results into interpretable network structures using igraph.
  • Classification of Errors: Computed color statistics for each region and trained a LogisticRegression model (scikit-learn) to detect segmentation mistakes.
  • Optimization: Profiled the pipeline with cProfile and snakeviz, achieving a 20√ó runtime improvement through memory management and algorithmic pruning.
  • Data Backtracking: Given fiber composition data at a specific frame, implemented an algorithm to backtrack and infer composition for prior frames ‚Äî enabling more targeted stimulation planning.

Results

  • Automated fiber detection reduced manual analysis time by ~8 hours per patient.
  • Achieved 20√ó performance speedup after profiling and optimization.
  • Provided a framework for visualizing nerve connectivity and predicting stimulation outcomes.
