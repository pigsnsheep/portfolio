Spotify Recommender System

Designed and implemented a personalized music recommendation engine using Spotify's Web API, and audio feature embeddings.

Problem

While Spotify's native recommendations are strong, they rely heavily on global user trends ("hits"). The goal of this project was to build a recommender that could not only suggest hits, but songs that users have previously never heard before and might enjoy. recommend

Approach

  • Data Collection: Starting with a list of popular songs, generated training data by prompting the user on whether or not they liked the song.
  • Feature Extraction: Extracted audio features (e.g., danceability, energy, valence, tempo) and embedded them into a multidimensional feature space.
  • Filtering: Determined which features the user valued, and used those features to train a classifier model.
  • Interface: Developed a simple frontend where users canreceive tailored recommendations.

Results

  • Generated recommendations closely aligned with user taste, validated through user testing.
  • Created interpretable visualizations showing the range of audio features the user likes.

Reflection

This project deepened my understanding of ML system design and filtering techniques. It reinforced how machine learning can be combined with human-centered design to deliver both accurate and explainable recommendations in real-world applications.
